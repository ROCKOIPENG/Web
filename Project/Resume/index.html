  4. 大连拓升科技有限公司实习期作品——基于STM32f103与ESP32-S3芯片的起重机多功能检测设备

    2024.07.15 — 2024.08.15

    项目简介：
    该项目是一款起重机多功能检测设备，它由STM32f103为主控的激光控制、自动旋转调平部分和ESP32-s3为主控的触摸屏两大部分组成，实现起重机相关功能属性的检测。

    实现功能：
    • 摄像头采集画面回传
    • 拱度测量
    • 挠度及静刚量测量
    • 制动下滑量测量
    • 位移速度测量
    • 空间中两点距离测量
    • 单点距离测量

    我负责的部分：
    我主要负责ESP32方面的开发，主要负责以下三大任务。
    1.LVGL液晶屏显示任务
    该任务用于在GT911电容触摸屏上实现TEXT、BUTTON、IMAGE、CHECKBOX、LED、LABEL、SWITCH、TABLE、WINDOWS、SLIDER和CHART等LVGL控件的布局和显示，以及相应的点击滑动事件监听、触发对应的回调函数等功能。
    2.OV2640摄像头采集任务
    该任务用于获取OV2640摄像头采集的画面信息，并经过字节交换等变换将画面传输到LVGL的image控件中，同时用LVGL的SLIDER控件可以调节摄像头画面的缩放大小。
    3.串口通信任务
    该任务用于通过制定好的串口通信协议与STM32底板进行串口通信，实现部分物理量如拱度，挠度，瞬时速度，距离的测量和开关激光等功能。

    硬件环境：
    ESP32-s3 N16R8

    开发环境：
    Visual Studio Code + 乐鑫ESP-IDF扩展插件

    编译环境：
    ESP-IDF V4.4.2

    调试环境：
    Open OCD

    结果：
    为公司原设备加上了可触摸的交互屏幕，让产品交互得到了大幅度的提升，大大地促进了销量。
